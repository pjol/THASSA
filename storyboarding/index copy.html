Hero Moment · 3 Panels

Minimal narrative: intent, verification, finality.

Panel 1

Intent

A requester opens demand for fresh oracle state.

Action

  • Requester funds and submits request/bid.
  • Hub records state and emits events.
placeBid(...) or submitManualUpdate(...)
Demand is now explicit on-chain state.
Panel 2

Verification

Node fulfills and hub runs strict checks.

Checks

  • query/shape/model commitments match oracle spec.
  • digest verifies via verifier module.
  • expiry + replay rules enforced.
verifyUpdate(digest, signedUpdate) → true
Only commitment-consistent payloads can pass.
Panel 3

Finality

Callback executes and economics settle.

Result

  • Client callback updates contract state.
  • Hub closes bid and marks digest consumed.
  • Fee, payout, and lockup settlement complete.
updateOracle(...) + token transfers
Data and incentives finalize in one deterministic flow.
